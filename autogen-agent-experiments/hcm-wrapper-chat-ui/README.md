# HCM Wrapper Chat UI

This folder contains a browser-based chat UI backed by an HCM wrapper or external completion service.

Files:
- `app.py`: current backend implementation.
- `app_old.py`: older backend variant retained for reference.
- `index.html`: current UI.
- `index_old.html`: older UI variant retained for reference.

Notes:
- Endpoint and credential values are placeholders in the checked-in code.
