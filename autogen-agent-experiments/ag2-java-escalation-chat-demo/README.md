# AG2 Java Escalation Chat Demo

This demo combines a Python AG2-backed service with a Java client.

Files:
- `server.py`: FastAPI service that routes chat requests through AG2 agents.
- `AG2ChatClient.java`: reusable Java client for calling the chat service.
- `ChatDemo.java`: simple Java entry point for exercising the client.

Notes:
- The checked-in code uses placeholder credentials and sample endpoints.
- Update the environment-backed values before running the Java client or server.
