# AG2 Supervisor Barge-In Chat

This demo shows a support chat where a supervisor can barge in when the conversation quality drops.

Files:
- `server.py`: FastAPI backend for the chat flow.
- `client.html`: browser client for exercising the demo.
- `requirements.txt`: Python dependencies.
- `run-instructions.sh`: helper script and local run notes.

Notes:
- Replace the sample endpoint and credentials placeholders before running it.
