# Google-Assisted HCM Wrapper Chat

This demo combines a chat service with a Google-assisted search step before sending the prompt to an external completion service.

Files:
- `app.py`: FastAPI and WebSocket backend.
- `googlesearch.py`: helper for Google Custom Search requests.
- `index.html`: browser client UI.

Notes:
- The checked-in Google and external API values are placeholders and must be set before use.
